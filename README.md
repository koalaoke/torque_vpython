# 
Simulando o torque de uma espira com vpython

# Executando o projeto

1. Crie um ambiente virtual
```bash
    python -m venv venv
    # Powershell
    .\env\Scripts\Activate.ps1
    # Linux
    source .\env\Scripts\activate
```
2. Instale os pacotes
```bash
    pip install -r requirements.txt
```

3. Rode main.py
```bash
    python main.py
```
